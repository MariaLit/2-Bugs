- 'Беларуская мова';
- 'Всемирная история';
- 'Искусство';
- 'Испанский язык';
- 'Китайский язык';
- 'Математика';
- 'Немецкий язык';
- 'Русская литература';
- 'Русский язык';
- 'Французский язык';
- 'Человек и мир'.
